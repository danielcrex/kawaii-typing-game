# ✨ Kawaii Typing Friends ✨

A cute, encouraging touch-typing game for kids. Adorable mascots fall gently from the top of the screen — type the letter or word on each tile before it reaches the bottom!

## How to play

Just open **`index.html`** in any web browser. No installation needed.

- Type the word shown on the lowest tile, then press **Enter** or **Space** to submit (single letters submit automatically).
- Clear **20 tiles** to complete a level.
- You have **3 hearts** — a tile that reaches the bottom costs one heart.
- Get **5+ correct in a row** for a sparkly power-up that slows time (and restores a heart at 10 in a row!).

## Features

- **24 levels**, each with its own kawaii mascot, from home-row single letters up to long words and short sentences.
- Gentle difficulty curve: fall speed, number of tiles on screen, and word length all increase gradually.
- Stars (1–3) awarded per level based on hearts remaining.
- Progress and stars saved automatically in your browser.
- Bright, calm pastel visuals with celebration sparkles and encouraging messages.
